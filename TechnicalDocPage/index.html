Introduction

This is a straigtforward summary about all you need to know about JavaScript. So let's get into it...

JavaScript is a powerful programming language commonly used in web development to add interactivity and dynamic behavior to websites. It was created by Brendan Eich in 1995 and has since become one of the core technologies of the World Wide Web. JavaScript enables developers to manipulate the content and behavior of web pages in real-time, allowing for features like form validation, animations, interactive maps, and much more.

As a client-side scripting language, JavaScript runs in the user's web browser, meaning that it can interact with the Document Object Model (DOM) of a web page to dynamically modify its structure, content, and style. Additionally, JavaScript can be used on the server-side through platforms like Node.js, expanding its capabilities beyond just browser-based applications.

JavaScript is known for its versatility and flexibility, as it can be integrated with HTML and CSS seamlessly. It is also supported by all major web browsers, making it a widely adopted choice for web development projects.

Data Types

JavaScript provides several data types to work with, including numbers, strings, booleans, arrays, objects, and more. These data types allow developers to store and manipulate different kinds of information within their programs. For example, numbers are used for mathematical calculations, strings for text manipulation, and booleans for logical operations.

In order to understand, JavaScript (the programming language that provides intercativity in the Internet), first you need to understand the data types and the variables of the language and how they are created. It would be really helpful when, where and how to use them.

Variables

Variables helps you remember the value of something for a later use in your code!

You can create a variable by following the syntax [keyword][name] . Example:

var aVariable;

The key word var says that a variable will be created, and aVariable is the name of the variable.

Work with variables in block scopes

Using the keyword let providess the block scope of variables. It is recommended to use it instead of var to better manage the scope application.

JavaScript uses different block scopes, such as, the global block scope and the function block scope, to determine which variables are available in which part of the code. As an example, a global variable is available inside a function because it belong to a global block scope. This is the case for the variables with the keyword var, they can be visible inside and outside a for loop, which may led to unexpected results. This is why let is the best option because it will available just in the for loop, so its availability could be more predictable.

This is the syntax for declaring a variable using let:

let myVariable;

Value assignment

myVariable = 123;

Now, myVariable has been assigned the value 123.

Explicit Initialization

Here a variable is being declare and a value has been assigned to it in the same line:

let myVariable = 123;

Constants

Sometimes you don't want to a variable to change its value in your program. This situation may ocurr when using mathematical constant, such as, pi. To declare and initialize a constant you can use the following syntax:

const MY_VARIABLE = 123;

Remember:
It is recommended to declare a constant with all capital letters!

Constants are similar to variables, with 2 exepctions:

  • They must have a value. Constants must be initialized, otherwise it will create an error.
  • The reference can not be changed. If the reference of a constant changed after being initiliazed it will create an error.

Let's see some examples:

  • This is not allowed:
    • const PI = 3;
    PI = 4;
  • The following syntax has a protected object reference and is not allowed:.
    • const obj = { a: 3 };
    obj = { b: 5 };
  • The next syntax has a object value that is not protected. Then, it is allowed becuase it will change the value of the object, but the reference itself.
    • const obj = { a: 3 };
    obj.a = 5;
Functions

Functions in JavaScript allow developers to encapsulate reusable blocks of code. They can accept parameters as input and return values as output, providing a way to organize and modularize code. Functions can be declared using the function keyword or defined as arrow functions for more concise syntax.

Making Decisions

Making decisions in JavaScript involves using conditional statements such as if, else if, and else. These statements allow developers to execute different blocks of code based on specified conditions. Conditional statements evaluate expressions to either true or false, determining which branch of code to execute.

Loops

Loops in JavaScript are used to repeat a block of code multiple times. The for loop, while loop, and do-while loop are common looping constructs used to iterate over arrays, objects, or perform tasks until a certain condition is met. Loops provide a way to automate repetitive tasks and process data efficiently.
